# Namaz Vakti

## React ile mobil uyumlu Namaz vakti web uygulaması

- Vakit gelince alarm çalar

- Sabah öğle akşama göre arkaplan resmi değişir

- Diğer vakte kalan süreyi gösterir

- Hicri takvimi gösterir

- Cuma günü hatırlatıcı
